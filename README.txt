Тестовый чат-бот ChatOps на API SberChat. Позволяет получать информацию о состоянии сервера, установок релизов и управлять DevOps трубой через SberChat.
Ниже описан функционал.
Получение информацию о состоянии сервера(uptime, запущенные процессы, статистика использования ресурсов, и т.д.).
Выполнение административные действия на сервере(установка новой версии АС, перезапуск процессов/сервера, очистка temp директорий, и т.д.).
Подписка на уведомления о статусе сборки, результатах выполнения тестов, результате установки АС и т.п.
Получение информацию о наличии новых commit/pull request в GIT и запустить сборщик релиза на указанном commit/pull request.

Для работы необходимо установить зависимости из requirements.txt и прописать токен для бота в системных переменных 'BOT_TOKEN' в
конфигурации запуска.
Запускать файл dialog_echo_bot.py .

Для корректной работы необходимо в конфигурации указать системные переменные BOT_TOKEN, JENKINS_USERNAME, JENKINS_PASSWORD.
